Sobre o jogo

Apresentação

Este jogo foi feito a partir de três tecnologias html, css e javascript. Toda a parte visual do jogo foi feito para ser o mais agradável possivel para que o jogador tenha uma experiencia visual confortável e ao mesmo tempo amigável, funcionando em dispositivos de várias resoluções, isto inclui dispositivos móveis.

Espero que goste do jogo e se divirta!!

Funcionamento

O jogo tem três níveis de dificuldade facíl, médio e difícil, os níveis se relacionam ao tempo que o jogador terá para resolver cada conta, sendo as mesmas geradas de forma aleátoria, então não se assuste com as contas. Falando em resposta o sistema de resposta é gerado de forma dinâmica a partir do javascript sendo um sistema de resposta objetivo onde só uma resposta será a correta.
Sobre o temporizador, ele será resetado a cada resposta que vc der com base no nível de dificuldade (como um reset de tempo caso a resposta seja correta).

OBS: uma vez o jogo sendo iniciado o botão de dificuldade será desabilitado até que o tempo acabe.

Placares

Irá haver três placares, sendo estes organizados pela dificuldade escolhida que irá ranquear os jogadores com base na quantidade de resposta respondidas corretamente, lembrando que não existe um limite de resposta, o jogo só acaba quando o tempo acaba
